START
  ↓
1. Browse Products (No auth needed)
   GET /api/products/featured
   GET /api/products/search?q=bamboo
   GET /api/categories
   ↓
2. Register New Account
   POST /api/auth/register
   {
     "name": "Test User",
     "email": "test@example.com",
     "password": "password123"
   }
   ↓
   Token saved automatically!
   ↓
   AUTHORISATION: Authorization: Bearer YOUR_TOKEN_HERE
3. Add Products to Cart (Auth required)
   POST /api/cart/items
   {
     "productId": "...",
     "quantity": 2
   }
   ↓
4. View Cart
   GET /api/cart
   ↓
5. Update Quantities
   PUT /api/cart/items/:itemId
   {
     "quantity": 3
   }
   ↓
6. Create Order
   POST /api/orders
   {
     "shippingAddress": {...},
     "paymentMethod": "card"
   }
   ↓
7. View Order History
   GET /api/orders
   ↓
8. Cancel Order (if needed)
   PATCH /api/orders/:orderId/cancel
   ↓
END
